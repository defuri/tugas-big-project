Belajar Javascript Untuk Pemula

Javascript adalah bahasa pemrograman memungkinkan Anda mengimplementasikan fungsi yang kompleks pada halaman web. Setiap kali halaman web melakukan lebih dari sekadar duduk di sana dan menampilkan informasi statis untuk Anda lihat-menampilkan pembaruan konten yang tepat waktu, peta interaktif, grafik animasi 2D/3D, kotak video bergulir, atau lebih banyak lagi-Anda dapat bertaruh bahwa JavaScript mungkin terlibat.

Modul

Kebijakan kami tentang JavaScript modern

JavaScript adalah bahasa yang terus berkembang dan telah banyak berubah selama bertahun-tahun. Khususnya, edisi ke-6 dari bahasa ini (kadang-kadang dikenal sebagai ECMAScript 2015 atau ES6), yang diperkenalkan pada tahun 2015, menambahkan banyak fitur baru. Pada saat yang sama, untuk menjaga kompatibilitas dengan situs web lama, fitur-fitur lama dari bahasa ini tetap dipertahankan, bahkan ketika fitur-fitur tersebut sudah tidak lagi dianggap sebagai praktik yang baik.

Kami berpikir bahwa fitur-fitur yang ditambahkan ke JavaScript di ECMAScript 2015 dan versi selanjutnya memungkinkan para pengembang untuk menulis kode yang lebih mudah dibaca, dapat diandalkan, dan ekspresif, dan penting untuk mempelajarinya.

Fitur-fitur yang kami ajarkan dalam kursus ini stabil dan telah didukung oleh semua browser utama selama beberapa tahun.

Langkah Pertama Javascript

Dalam modul JavaScript pertama kami, pertama-tama kami akan menjawab beberapa pertanyaan mendasar seperti "apa itu JavaScript?", "seperti apa JavaScript itu?", dan "apa yang dapat dilakukannya?", sebelum melanjutkan untuk membawa Anda melalui pengalaman praktis pertama Anda dalam menulis JavaScript. Setelah itu, kita akan membahas beberapa fitur utama JavaScript secara detail, seperti variabel, string, angka, dan array.

Blok Pembangun Javascript

Dalam modul ini, kita akan melanjutkan cakupan semua fitur dasar utama JavaScript, dengan mengalihkan perhatian kita ke jenis blok kode yang sering ditemui seperti pernyataan bersyarat, perulangan, fungsi, dan peristiwa. Anda telah melihat hal-hal ini dalam kursus, tetapi hanya secara sepintas - di sini kita akan membahas semuanya secara eksplisit.

Memperkenalkan Objek Javascript

Dalam JavaScript, hampir semua hal adalah objek, mulai dari fitur inti JavaScript seperti string dan array hingga API browser yang dibangun di atas JavaScript. Anda bahkan dapat membuat objek Anda sendiri untuk merangkum fungsi dan variabel terkait ke dalam paket yang efisien. Sifat JavaScript yang berorientasi objek sangat penting untuk dipahami jika Anda ingin melangkah lebih jauh dengan pengetahuan Anda tentang bahasa ini dan menulis kode yang lebih efisien, oleh karena itu kami menyediakan modul ini untuk membantu Anda. Di sini kami akan mengajarkan teori objek dan sintaksis secara detail, melihat cara membuat objek Anda sendiri, dan menjelaskan apa itu data JSON dan bagaimana cara menggunakannya.

Javascript Asinkron

Dalam modul ini kita akan membahas JavaScript asinkron, mengapa JavaScript asinkron itu penting, dan bagaimana JavaScript asinkron dapat digunakan untuk menangani potensi pemblokiran secara efektif, seperti mengambil sumber daya dari server.

Web API Sisi Klien

Ketika menulis JavaScript sisi klien untuk situs web atau aplikasi, Anda tidak akan melangkah lebih jauh sebelum mulai menggunakan API - antarmuka untuk memanipulasi berbagai aspek peramban dan sistem operasi tempat situs tersebut berjalan, atau bahkan data dari situs web atau layanan lain. Dalam modul ini kita akan mempelajari apa itu API, dan bagaimana cara menggunakan beberapa API yang paling umum yang akan sering Anda temui dalam pekerjaan pengembangan Anda.

Menyelesaikan Masalah Umum Javascript

Memecahkan masalah umum dalam kode JavaScript Anda memberikan sedikit saran tentang cara menghindari kesalahan pemrograman JavaScript pemula yang umum, bersama dengan banyak tautan bermanfaat ke topik-topik yang menunjukkan cara memecahkan masalah pemrograman JavaScript yang umum.
