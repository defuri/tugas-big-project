Tutorial

Tautan di halaman ini mengarah ke berbagai tutorial dan materi pelatihan. Baik Anda baru memulai, mempelajari dasar-dasarnya, atau sudah berpengalaman dalam pengembangan web, Anda bisa menemukan sumber daya yang bermanfaat di sini untuk berlatih.

Sumber daya ini dibuat oleh perusahaan dan pengembang web yang berpikiran maju yang telah menggunakan standar terbuka dan praktik terbaik untuk pengembangan web dan yang menyediakan atau mengizinkan penerjemahan, melalui lisensi konten terbuka seperti Creative Commonns.

Untuk orang awam yang baru mengenal Web

Memulai dengan Web
Memulai dengan Web adalah seri ringkas yang memperkenalkan Anda pada dasar-dasar pengembangan web. Anda akan menyiapkan alat yang Anda butuhkan untuk membuat halaman web sederhana dan mempublikasikan aplikasi Anda sendiri.

Tutorial HTML

Level Pemula

Pengenalan HTML

Modul ini akan membiasakan Anda dengan konsep dan sintaks yang penting, melihat penerapan HTML pada teks, cara membuat hyperlink, dan cara menggunakan HTML untuk menyusun halaman web.

Membuat Website Sederhana dengan HTML

Panduan HTML untuk pemula yang mencakup penjelasan tentang tag-tag umum, termasuk tag HTML. Juga termasuk panduan langkah demi langkah untuk membuat halaman web dasar dengan contoh kodenya.

Tantangan HTML

Gunakan tantangan ini untuk mengasah kemampuan HTML Anda (misalnya, "Haruskah saya menggunakan elemen <h2> atau elemen <strong>?"), dengan fokus pada tag yang bermakna.</p>

Level Menengah

Multimedia dan Embedding

Modul ini membahas cara menggunakan HTML untuk menyertakan multimedia di halaman web Anda, termasuk berbagai cara untuk menyertakan gambar, dan cara meng-embed video, audio, dan bahkan seluruh halaman web lainnya.

Tabel HTML

Merepresentasikan data tabel pada halaman web dengan cara yang mudah dimengerti dan mudah diakses dapat menjadi sebuah tantangan. Modul ini mencakup markup tabel dasar, bersama dengan fitur yang lebih kompleks seperti mengimplementasikan keterangan dan ringkasan.

Level Lanjutan

Form HTML

Formulir adalah bagian yang sangat penting dari Web - formulir ini menyediakan banyak fungsi yang Anda butuhkan untuk berinteraksi dengan situs web, misalnya mendaftar dan masuk, mengirim umpan balik, membeli produk, dan banyak lagi. Modul ini membantu Anda memulai dengan membuat bagian sisi klien dari formulir.

Tips untuk membuat halaman HTML yang loadingnya cepat

Optimalkan halaman web untuk menyediakan situs yang lebih responsif bagi pengunjung dan mengurangi beban pada server web dan koneksi Internet Anda.

Tutorial CSS

Level Pemula

Dasar CSS

CSS (Cascading Style Sheets) adalah kode yang Anda gunakan untuk menata halaman web Anda. Dasar-dasar CSS akan memandu Anda tentang apa yang Anda perlukan untuk memulai. Kami akan menjawab pertanyaan-pertanyaan seperti: Bagaimana cara membuat teks menjadi hitam atau merah? Bagaimana cara membuat konten saya muncul di tempat tertentu di layar? Bagaimana cara menghias halaman web saya dengan gambar dan warna latar belakang?

Langkah Pertama CSS

CSS (Cascading Style Sheets) digunakan untuk mendesain dan menata halaman web - misalnya, untuk mengubah font, warna, ukuran, dan spasi konten Anda, membaginya ke dalam beberapa kolom, atau menambahkan animasi dan fitur dekoratif lainnya. Modul ini memberikan awal yang baik untuk Anda dalam menguasai CSS dengan dasar-dasar cara kerjanya, seperti apa sintaksisnya, dan bagaimana Anda dapat mulai menggunakannya untuk menambahkan gaya pada HTML.

Blok Penyusun CSS

Modul ini melanjutkan apa yang telah dipelajari pada modul sebelumnya - setelah Anda mengenal bahasa dan sintaksisnya, serta memiliki pengalaman dasar dalam menggunakannya, sekarang saatnya untuk mempelajari lebih dalam lagi. Modul ini akan membahas tentang cascade dan inheritance, semua tipe selector yang tersedia, unit, ukuran, menata latar belakang dan batas, debugging, dan masih banyak lagi.

Tujuannya adalah untuk memberikan Anda alat untuk menulis CSS yang kompeten dan membantu Anda memahami semua teori yang penting, sebelum beralih ke disiplin ilmu yang lebih spesifik seperti penataan teks dan tata letak CSS.

Selector

Menargetkan elemen HTML, termasuk berdasarkan kondisi elemen, dengan CSS.

Spesifisitas

Memahami algoritma browser untuk menentukan deklarasi CSS mana yang akan diterapkan pada sebuah elemen ketika ada deklarasi yang bersaing

Cascade & Inheritance

Caskade, khususan, dan inheritance mengontrol bagaimana CSS diterapkan pada HTML dan bagaimana konflik antara deklarasi elemen diselesaikan.

Menghias Text

Di sini kita akan membahas dasar-dasar styling teks, termasuk pengaturan font, tebal, dan miring, spasi baris dan huruf, serta bayangan dan fitur teks lainnya. Kami mengakhiri modul ini dengan melihat penerapan font khusus ke halaman Anda, dan menata list dan tautan.

Pertanyaan Umum CSS

Pertanyaan umum dan jawaban untuk pemula

Level Menengah

CSS Layout

Pada titik ini kita telah melihat dasar-dasar CSS, cara menggayakan teks, dan cara menggayakan dan memanipulasi kotak tempat konten Anda berada. Sekarang saatnya untuk melihat bagaimana menempatkan box Anda di tempat yang tepat dalam kaitannya dengan viewport, dan satu sama lain. Kita telah membahas prasyarat yang diperlukan sehingga sekarang kita dapat menyelami tata letak CSS, melihat pengaturan tampilan yang berbeda, metode tata letak tradisional yang melibatkan float dan pemosisian, dan alat tata letak bermodel baru seperti flexbox.

Referensi CSS

Referensi lengkap untuk CSS, dengan rincian tentang dukungan oleh Google Chrome dan peramban lainnya.

Fluid Grids

Desain tata letak yang dapat diubah ukurannya secara lancar dengan jendela browser, namun tetap menggunakan kisi-kisi tipografi.

Tantangan CSS

Kembangkan kemampuan CSS Anda, dan lihat di mana Anda membutuhkan lebih banyak latihan.

Level Lanjutan

Menggunakan CSS transforms

Menerapkan rotation, skewing, scaling, and translation menggunakan CSS.

Transisi CSS

Transisi CSS menyediakan cara untuk menghidupkan perubahan pada properti CSS, alih-alih membuat perubahan tersebut berlaku secara instan.

Tutorial Javascript

Level Pemula

Langkah Pertama Javascript

Dalam modul JavaScript pertama kami, pertama-tama kami akan menjawab beberapa pertanyaan mendasar seperti "apa itu JavaScript?", "seperti apa JavaScript itu?", dan "apa yang dapat dilakukannya?", sebelum melanjutkan untuk membawa Anda melalui pengalaman praktis pertama Anda dalam menulis JavaScript. Setelah itu, kita akan membahas beberapa fitur utama JavaScript secara detail, seperti variabel, string, angka, dan array.

Block Pembangun Javascript

Dalam modul ini, kita akan melanjutkan cakupan semua fitur dasar utama JavaScript, dengan mengalihkan perhatian kita ke jenis blok kode yang sering ditemui seperti pernyataan bersyarat, perulangan, fungsi, dan event. Anda telah melihat hal-hal ini dalam kursus, tetapi hanya secara sepintas - di sini kita akan membahas semuanya secara gamblang.

Memulai Dengan Javascript

Apa itu javascript dan bagaimana itu bisa membantumu?

Level Menengah

Perkenalan Objek Javascript

Dalam JavaScript, hampir semua hal adalah objek, mulai dari fitur inti JavaScript seperti string dan array hingga API browser yang dibangun di atas JavaScript. Anda bahkan dapat membuat objek Anda sendiri untuk merangkum fungsi dan variabel terkait ke dalam paket yang efisien. Sifat JavaScript yang berorientasi objek sangat penting untuk dipahami jika Anda ingin melangkah lebih jauh dengan pengetahuan Anda tentang bahasa ini dan menulis kode yang lebih efisien, oleh karena itu kami menyediakan modul ini untuk membantu Anda. Di sini kami akan mengajarkan teori objek dan sintaksis secara detail, melihat cara membuat objek Anda sendiri, dan menjelaskan apa itu data JSON dan bagaimana cara menggunakannya.

API Sisi Klien

Ketika menulis JavaScript sisi klien untuk situs web atau aplikasi, Anda tidak akan melangkah lebih jauh sebelum mulai menggunakan API - antarmuka untuk memanipulasi berbagai aspek browser dan sistem operasi tempat situs berjalan, atau bahkan data dari situs web atau layanan lain. Dalam modul ini, kita akan mempelajari apa itu API, dan bagaimana cara menggunakan beberapa API yang paling umum yang akan sering Anda temui dalam pekerjaan pengembangan Anda.
